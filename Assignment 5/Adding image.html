1. JPEG and JPG It’s important to note that JPEG and JPG are the same file formats, with different acronyms and file extensions. Joint Photographic Experts Group (JPEG) is a raster image with lossy compression. Its lossy compression means that JPEG eliminates some data to reduce its file size, which decreases its image quality in return. Its relatively small file sizes allow you to save more space on your disk or memory card.
